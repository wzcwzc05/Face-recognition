使用说明:
1.将待识别的照片放入unknown-picture文件夹中
2.如果有想加入数据库的人脸图片请放入known-picture文件夹中，
每次识别前都会预处理这些照片
PS：每张待加入的照片的文件名即被认为是该照片的人名（只能包含一张人脸）
3.最后程序正常退出后所有处理的照片全以原文件名输出在output文件夹中
*******************************************************
config.ini 配置说明
RectangleColour代表框的颜色
FontLocation代表字体位置
FontColour代表字体颜色
FontSize代表字体大小
